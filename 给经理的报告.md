## 🚀 快速部署命令

如果您想立即测试，以下是完整的部署流程：

### 在您的101服务器上：
```bash
# 1. 克隆代码（或拉取最新）
cd ~/Teleop_platform
git pull

# 2. 查找USB端口
ls /dev/tty* | grep -E "USB|ACM"

# 3. 启动服务器
python run_server.py --backend soarm --soarm-port /dev/ttyUSB0 --host 0.0.0.0 --port 8000
```

### 在美国的电脑上：
```bash
# 访问浏览器
http://服务器IP:8000

# 或运行延迟测试
python client/latency_test_client.py --server ws://服务器IP:8000/ws/v1/teleop --samples 200
```
